## 우리 팀 창업 아이디어 정리

# AI 기반 직무 추천·가상 체험 플랫폼

1. 아이디어 개요

중고등학생들은 진로 정보 부족과 직무 이해 부족으로 인해 자신에게 맞는 진로를 선택하는 데 어려움을 겪고 있음.
이에 따라 우리는 AI 기반 직무 추천 + 직무 가상 체험 기능을 결합한 새로운 진로 탐색 서비스를 기획했습니다.

⸻

2. 문제 인식
	•	학생들은 직무 정보에 직접적으로 접근하기 어렵고, 설명만으로는 실제 업무를 이해하기 어렵습니다.
	•	직무에 대한 충분한 이해 없이 전공과 진로를 선택하여 이후 불일치 문제(전공 후회, 직업 불만족)가 자주 발생합니다.

⸻

3. 해결책

AI 기반 직무 추천 기능
	•	학생이 입력하는 정보
	•	흥미 키워드
	•	보유 역량
	•	선호 전공
	•	AI가 종합 분석하여 학생에게 적합한 직무를 추천

직무별 가상 업무 시뮬레이션 (핵심 차별점)
	•	단순 정보 제공이 아니라 학생이 직접 체험할 수 있는 형태
	•	예시:
	•	UX 디자이너 → 간단한 와이어프레임 제작 체험
	•	기획자 → 문제 정의 → 해결 방안 선택 → 기획안 작성 미션
	•	개발자 → 기초 알고리즘 구조 선택, 오류 수정 게임
	•	마케팅 → 타깃 분석 후 캠페인 기획 미션

→ 설명만으로는 이해하기 어려운 직무를 직접 경험하며 적성 확인 가능

⸻

4. 비즈니스 모델 (BM)

교육 기관 대상 B2G 라이선스 판매 모델

판매 대상
	•	교육부
	•	시·도 교육청
	•	진로교육지원센터
	•	중·고등학교 진로부서

방식
	•	기관 단위로 플랫폼 라이선스를 구매
	•	학생들은 학교 계정으로 접속해 직무 추천/체험 이용
	•	교사용 대시보드를 통해 학생 진로 탐색 현황 조회 가능

기대 효과
	•	기관: 진로 교육 강화, 직무 미스매치 예방
	•	우리 팀: 안정적인 B2G 수익 모델 확보

⸻


5. 기대 효과

교육·사회적 효과
	•	학생의 진로 미스매치를 줄여 국가 인력 효율성 향상
	•	자신에게 맞는 진로를 조기에 발견하여 불안 감소

정서적 효과
	•	“직접 해보는 체험”을 통해 진로 불확실성 해결
	•	진로 탐색에 대한 자신감 향상

⸻

7. 팀 구성
	•	총 4명
	•	학교 마케팅 동아리 WBS와 연합 운영
	•	초기 사용자 조사, 홍보, 실제 사용자 확보 프로세스를 공동 진행

⸻

8. 참가 후기

이번 캠프를 통해 단순한 아이디어 제안이 아니라, 실제 시장성·현실성·사업 구조까지 고민해볼 수 있었습니다.
멘토링 과정에서 특히 “학교와 교육 기관 대상 B2G 모델”의 중요성을 알게 되었고, 우리 서비스의 확장성과 차별점을 더 명확히 정리할 수 있었습니다.

직무 추천뿐 아니라 시뮬레이션을 통한 직무 체험 기능이 큰 강점이 될 수 있다는 점도 확인했습니다.
이번 경험은 아이디어를 실제 서비스로 발전시키는 데 중요한 기반이 되었고, 앞으로 실제 Demo와 MVP를 만드는 데 도움을 줄 수 있는 방향성을 얻을 수 있었습니다.
